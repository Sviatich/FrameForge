# FrameForge

FrameForge - веб-приложение для преобразования выбранного Figma frame в статический HTML/CSS-проект.

Пользователь подключает Figma через OAuth, вставляет ссылку на макет, выбирает нужный frame и получает предпросмотр, исходный код и ZIP-архив с результатом.

## Стек

- **Next.js** - основа приложения.
- **React** - интерфейс загрузки макета, выбора frame, предпросмотра и просмотра кода.
- **TanStack Query** - клиентская работа с состоянием запросов и OAuth-сессии.
- **Zod** - проверка входных и выходных данных API.
- **Figma REST API** - загрузка структуры файла, метаданных и ссылок на изображения.
- **JSZip** - сборка результата в ZIP-архив.

## Пайплайн

1. Пользователь подключает Figma через OAuth и вставляет ссылку на макет.
2. Приложение получает структуру Figma-файла и собирает список доступных frames.
3. Пользователь выбирает нужный frame для преобразования.
4. Сервер загружает данные выбранного frame и связанные изображения.
5. Parser переводит Figma JSON во внутреннюю нормализованную модель.
6. Transformer преобразует эту модель в структуру будущей HTML/CSS-верстки.
7. Generator создает файлы проекта и данные для предпросмотра.
8. Пользователь просматривает результат и экспортирует проект в ZIP-архив.
